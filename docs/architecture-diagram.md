| Component | Tool Chosen | Alternatives Considered | Rationale |
|-----------|-------------|------------------------|-----------|

| **Orchestration** | Airflow | Prefect, Dagster | Industry standard, rich ecosystem |

| **Containerization** | Docker | - | Standard |
